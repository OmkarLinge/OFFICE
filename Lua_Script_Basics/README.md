# 🧩 Lua Basics

This folder contains beginner-friendly Lua programs designed to build a strong foundation in Lua scripting.

The goal of this section is simple:

> ✅ Learn Lua syntax  
> ✅ Understand programming logic  
> ✅ Practice real input-based programs  
> ✅ Prepare for embedded systems and UAV scripting

All programs are written in a clean and easy-to-understand format so that anyone new to Lua can follow along.

---

## 📚 Learning Flow

The files are arranged in a logical order.  
Follow them sequentially for best understanding.



---

## 📂 File Descriptions

### ✅ Hello.lua
Introduces:
- Lua syntax
- Printing output
- Variables
- Basic data storage

You learn how a Lua program starts and runs.

---

### ✅ Operators.lua
Covers:
- Arithmetic operations
- Addition, subtraction, multiplication, division
- Working with numeric input

Shows how Lua performs calculations.

---

### ✅ Conditional_loops.lua
Introduces decision making using:
- `if`
- `elseif`
- `else`

Program behavior changes based on user input.

---

### ✅ for_loop.lua
Demonstrates:
- Controlled repetition
- Loop counters
- Automatic iteration

Used when number of repetitions is known.

---

### ✅ while_loop.lua
Shows:
- Condition-based looping
- Manual control of execution

Useful when repetition depends on conditions.

---

### ✅ user_input.lua
Teaches:
- Taking real-time input
- Interactive programs
- Runtime data handling

Programs respond to user input dynamically.

---

### ✅ data_types.lua
Explains Lua data types:
- String
- Number
- Boolean
- Nil

Helps understand how Lua stores information.

---

### ✅ functions.lua
Introduces:
- Function creation
- Parameters
- Returning values

Used to organize reusable logic.

---

### ✅ tables.lua
One of the most important Lua concepts.

Tables act as:
- Arrays
- Objects
- Dictionaries

Widely used in real Lua scripting systems.

---

### ✅ string_handling.lua
Covers text operations:
- String length
- Uppercase conversion
- Text processing

Useful for logs and command handling.

---

### ✅ math_operations.lua
Demonstrates Lua math library:
- Square root
- Rounding
- Numeric calculations

Used in real computational logic.
